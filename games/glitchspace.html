Glitchspace

Description


Glitchspace is a first person programming game being developed for Windows and MacOS. Currently in alpha - the game is centred around the use of a programming mechanic, which allows you to affect the world in various ways.

Glitchspace currently supports the Oculus Rift, and we have been looking at second screen support for a Playstation/Vita version.
