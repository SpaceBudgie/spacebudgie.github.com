Light in the Night

Description


Light in the Night - A first person experimental, exploration game created by Space Budgie for the 2014 Global Game Jam with a focus on light pollution. It's best played using the Oculus Rift, although it is not required. If you wish to use a Rift (provided you have one!) select it in the Graphics Quality option in the launcher. Most of the artwork is courtesy of Alastair Low.

Frustrated at how the night sky can not be admired in all of its beauty, we decided to create a game that tries to give an impression of what could be hidden behind all of that light that floods the sky.
